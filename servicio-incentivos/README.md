# n8n — Comandos
 
| Comando | Descripción |
|---|---|
| `make up` | Levanta el contenedor |
| `make down` | Detiene el contenedor (el volumen no se elimina) |
| `make deploy` | Levanta el servicio e importa los workflows desde `./workflows` |
| `make import-workflows` | Importa workflows sin reiniciar el contenedor |
| `make export-workflows` | Exporta los workflows activos hacia `./workflows` |
| `make save` | Exporta los workflows y los commitea |
